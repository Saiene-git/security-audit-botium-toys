# 🔐 Security Audit Simulation - Botium Toys

Este projeto contém uma simulação de auditoria de segurança realizada como parte do curso de cibersegurança do Google.

## 📄 Conteúdo
- Avaliação de riscos
- Identificação de vulnerabilidades
- Recomendações de segurança

## 🛠️ Principais pontos abordados
- Princípio do menor privilégio
- Criptografia de dados
- Backup e recuperação de desastres
- Políticas de senha
- IDS (Intrusion Detection System)

## 📌 Objetivo
Demonstrar habilidades iniciais em análise de segurança, identificação de riscos e elaboração de recomendações.
